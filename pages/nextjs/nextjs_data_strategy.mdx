# List of the nextjs Data Strategy 


## Static Page 

<ul>
<li>No Data from db</li>
<li>Static</li>
</ul>

## SSG (Static Site Generation)

<ul>
<li>data from db</li>
<li>Generate data when the app build</li>
<li>dosen't update (data don't change)</li>
</ul>

## SSR (Server Side Rendering)

<ul>
<li>Fast</li>
<li>better for seo</li>
<li>dosen't work for every request</li>
<li>Log between pages and js functionality not work</li>
</ul>

## ISR (Incremental static regeneration)

<ul>
<li>Reneder Static</li>
<li>Cased server</li>
<li>Cash update interval</li>
</ul>

## CSR (Client-Side Rendering)

<ul>
<li>Client fetch data from server</li>
<li>can use server state mangement (SWR)</li>
<li>Seo no require </li>
</ul>
